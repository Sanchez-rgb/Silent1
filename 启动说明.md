# 小红书情感分析平台 v2.0 - 启动说明

## 📋 环境准备

### 1. 安装 Python 依赖

```bash
pip install -r requirements.txt
```

### 2. 安装 Redis

#### Windows:
```bash
# 方式一：使用 Docker（推荐）
docker run -d -p 6379:6379 --name redis redis:latest

# 方式二：使用 Redis for Windows
# 下载地址：https://github.com/microsoftarchive/redis/releases
```

#### macOS:
```bash
brew install redis
brew services start redis
```

#### Linux:
```bash
# Ubuntu/Debian
sudo apt-get install redis-server
sudo systemctl start redis
```

## 📖 使用流程

### 1. 注册/登录

1. 访问前端页面
2. 注册一个新用户（或者使用已有的账号）
3. 登录系统

### 2. 配置用户信息

1. 在「用户设置」区域填入你的小红书博主ID
2. **可选但推荐**：填入你从小红书网页复制的 Cookie（可以大幅提升爬取成功率，减少反爬）
3. 点击「保存设置」

### 3. 启动爬虫任务

1. 点击「🚀 开始爬取我的所有笔记」按钮
2. 系统会立即返回 task_id 并开始异步爬取
3. 页面会自动轮询并显示进度条

### 4. 查看结果

- 任务完成后，点击「📊 查看报告」查看 HTML 报告
- 或者在下方查看仪表盘统计数据
- 或者在笔记列表中查看爬取到的笔记

## 🔧 常见问题

### Q: 提示 Redis 连接失败？

A: 检查 Redis 服务是否启动，确认端口 6379 是否被占用。

### Q: Celery 启动报错？

A: 确认当前目录是否正确，以及是否已安装所有依赖。

### Q: 爬虫任务一直失败？

A: 
1. 检查 Cookie 是否过期
2. 检查博主ID是否正确
3. 查看任务错误信息

### Q: 情感分析模型下载慢？

A: 
- 首次运行会自动下载约 1.5GB 的模型文件
- 需要稳定的网络连接
- 下载完成后会缓存到 `./model_cache` 目录
