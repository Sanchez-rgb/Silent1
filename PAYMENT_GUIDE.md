# 小红书AI文案生成器 - 付费系统使用指南

## 📋 功能概览

本系统实现了完整的付费VIP会员系统，包括：

1. **激活码系统** - 管理员生成激活码，用户输入激活码开通VIP
2. **扫码支付** - 展示微信/支付宝收款码，用户付款后联系客服激活
3. **VIP管理** - 完整的激活码管理和用户VIP状态追踪

## 🎯 用户使用流程

### 方式一：激活码开通

1. 用户登录系统
2. 点击"升级 VIP"按钮
3. 输入激活码
4. 点击"激活"按钮
5. VIP开通成功，有效期30天

### 方式二：扫码支付

1. 用户登录系统
2. 点击"升级 VIP"按钮
3. 选择支付方式（微信/支付宝）
4. 显示收款码截图提示
5. 用户扫码支付
6. 用户点击"联系客服"按钮
7. 复制客服微信，添加好友
8. 客服确认收款后，手动提供激活码

## 👨‍💼 管理员功能

### 访问管理后台

打开浏览器访问：http://localhost:8000/admin

### 管理功能

1. **查看统计**
   - 总激活码数量
   - 已使用激活码
   - 未使用激活码
   - VIP用户总数

2. **生成激活码**
   - 输入要生成的数量（1-100）
   - 点击"生成激活码"按钮
   - 激活码自动复制到剪贴板

3. **管理激活码**
   - 查看所有激活码
   - 查看激活码使用状态
   - 查看使用者手机号
   - 复制未使用的激活码

## 🔧 API 接口

### 用户API

#### POST `/api/activate`
使用激活码开通VIP

**请求体：**
```json
{
  "user_id": 123,
  "code": "VIPXXXXXXXXXXXXX"
}
```

**响应：**
```json
{
  "success": true,
  "message": "激活成功！VIP有效期至2024年12月31日",
  "expire_at": "2024-12-31T12:00:00"
}
```

### 管理员API

#### POST `/api/admin/generate-codes`
生成激活码

**请求体：**
```json
{
  "count": 10
}
```

**响应：**
```json
{
  "success": true,
  "codes": ["VIPXXXXXXXXXXXXX", "VIPXXXXXXXXXXXXX", ...],
  "message": "成功生成 10 个激活码"
}
```

#### GET `/api/admin/codes`
查看所有激活码

**响应：**
```json
[
  {
    "id": 1,
    "code": "VIPXXXXXXXXXXXXX",
    "is_used": false,
    "used_by_phone": null,
    "used_at": null,
    "created_at": "2024-01-01T12:00:00",
    "expires_at": "2024-01-31T12:00:00"
  }
]
```

## 💡 运营建议

### 1. 激活码管理

- 定期生成新的激活码
- 监控激活码使用情况
- 及时补充库存

### 2. 客服流程

1. 用户付款后，提供激活码
2. 或者在管理后台直接修改用户VIP状态：
   ```sql
   UPDATE users SET is_vip = 1 WHERE phone = '用户手机号';
   ```

### 3. 支付收款码

当前系统使用手动客服激活方式。如果需要自动化支付：

#### 推荐方案

**方案一：Stripe + 自动化**
- 集成 Stripe Checkout
- Webhook 回调自动发送激活码
- 需要：Stripe 账户（支持国际信用卡）

**方案二：Paddle**
- 支持支付宝、微信支付
- 订阅制 SaaS 支付平台
- 需要：Paddle 商家账户

**方案三：微信支付/支付宝直连**
- 需要商户号
- 开发成本高
- 手续费低
